BookIt!

Description

Book price aggregation app. Uses Google Books API, New York Times API, and GoodReads API.

Features

  • On-demand book price aggregation
  • Easy to use UI
  • OCR to scan book titles on the fly
  • Bookmark favourite books

Technologies

Language

  • Java

Libraries

  • Android
  • Firebase
  • Tesseract OCR
